# e-plantShopping

## Plant eShoppping App
This plant shopping app is built on React + Redux and is the final for the React course under the IBM Fullstack Developer Certification in Coursera

## Features
* Landing page
* Product view page
    * view available plants in individual cards with their product information
    * add plants to the cart for checkout
* Checkout page
    * view the plants that were added to the cart
    * adjust the quantity for each plant
    * remove plants from the cart
    * view the subtotal for each plant as well as the total for the cart
